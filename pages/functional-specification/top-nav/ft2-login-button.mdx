# 로그인 버튼 (FT2)

| ID    | FT2                                           |
|-------|-----------------------------------------------|
| 화면    | 상단 네비                                         |
| 세부 위치 | 우측 끝                                          | 
| EPIC  | 인증                                            |
| 기능 이름 | 로그인 버튼                                        |
| 설명    | 로그인 버튼을 통하여 Allqueue 서비스를 사용하기 위한 인증을 거칠 수 있다 |
| 개발 명세 | - 로그인 버튼을 클릭하여 “로그인 페이지로 이동”이 가능하다<br />- 어떤 화면에서든 로그인 버튼을 사용할 수 있어야한다.<br />- 현재 로그인이 되어있지 않은 상태여야 로그인 버튼이 활성화된다.<br />- 로그인이 되어있다면 로그아웃 버튼으로 변경된다.            |
| 구현 여부 |  |
| 디자인 여부 | O |

# 게임 매칭 버튼 (FT5)

| ID | FT5 |
| --- | --- |
| 화면 | 홈 |
| 세부 위치 | 홈화면 상단 중심부 |
| EPIC | 게임 매칭 |
| 기능 이름 | 게임 매칭 버튼 |
| 설명 | 게임 매칭 버튼을 클릭하여, 선택된 게임에 대한 매칭을 진행할 수 있다. |
| 개발 명세 | - 버튼 클릭 시 게임 매칭 플로우가 시작된다.<br />&emsp;a. 선택된 게임이 “모드 선택” 이 가능한 게임이라면 “모드 선택” 단계로 이동한다.<br />&emsp;b. 선택된 게임이 “모드 선택” 이 불가능한 게임이라면 바로 팀 매칭이 시작된다.<br />- 클릭 시 “홈” 페이지로 이동한다.<br />- 로그인 여부와 관계없이 활성화된다.<br />- 로그인이 안되어있는 상태로 버튼을 클릭하면 소셜 로그인페이지 로 이동한다. |
| 구현 여부 |  |

| ID | FT6 |
| --- | --- |

| EPIC | 플레이어 검색 |
| 기능 이름 | 플레이어 리스트 |
| 설명 | 플레이어 검색을 통해 검색된 사용자들의 리스트를 보여준다. |
| 개발 명세 | - 현재 선택된 게임의 플레이어 리스트를 간소화된 프로필 카드로 보여준다.<br />- “플레이어 검색” 항목이 비어있을 경우 랜덤으로 플레이어들을 선택하여 보여준다.<br />- 서버에 컴포넌트를 생성하기 위한 MetaData API를 요청하여 컴포넌트를 그린다.<br />&emsp;a. API를 통하여 리스트의 크기를 획득한다.<br />- “플레이어 검색” 결과가 있다면 그에 맞춰 프로필 카드를 노출한다.<br />- 검색 결과를 랜더링하는 동안 갱신 UI를 표시한다.(ex. 스피너) |

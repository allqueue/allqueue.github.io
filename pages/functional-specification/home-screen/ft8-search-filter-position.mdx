# 검색 필터 - 선호 포지션 (FT8)

| ID | FT8 |
| --- | --- |
| 화면 | 홈 |
| 세부 위치 | 홈화면 하단 |
| EPIC | 플레이어 검색 |
| 기능 이름 | 플레이어 검색 필터 - 선호 포지션 |
| 설명 | 플레이어 검색시 선택한 선호도에 해당하는 플레이어들을 검색한다. |
| 개발 명세 | - 플레이어 검색 컴포넌트 생성을 위한 API 를 호출하여 컴포넌트를 생성한다.<br />- API 메타데이터로부터 획득한 아이템 리스트를 드롭박스 형식으로 단일 선택 가능하다.<br />- 선택된 값은 플레이어 검색 시 필터 파라미터로 사용된다. |

# Match Click → 프로필 입력 (FT12)

| ID | FT12 |
| --- | --- |
| 화면 | Match Click → 프로필 입력 |
| 세부 위치 |  |
| EPIC | 회원가입 |
| 기능 이름 | Match Click → 프로필 입력 |
| 설명 | “매치 버튼”을 클릭하였으나 해당 게임에 대한 프로필이 등록되어있지 않았을 경우 사용자 프로필을 입력하도록 한다. |
| 개발 명세 | - Match Click 시 사용자 프로필이 등록되어있지 않았을 경우에만 해당 페이지로 이동되어야 한다.<br />- 일반적인 프로필 화면과 다르게 게임 선택이 불가능해야한다 Match Click 시 정해진 게임으로 고정<br />- 페이지 이동시 입력 받아야할 정보들은 API를 통해서 얻어와야 한다.<br />- 저장 버튼 클릭 시 Match 가 이어서 진행되어야 한다. |
| 구현 여부 |  |

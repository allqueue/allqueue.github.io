# 새 프로필 작성 (FT11)

| ID          | FT11                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 화면        | 새 프로필 작성                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| 세부 위치   |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |

| 기능 이름   | 새 프로필 작성                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| 설명        | 새로운 프로필을 작성한다.                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| 개발 명세   | - 플레이어 이름을 입력하면 유효성 검사를 진행한다.(20자 내, 특수문자 불가) <br/> - 티어 api를 불러와 티어를 선택한다. <br/> - 포지션 api를 불러와 포지션을 선택한다. <br/> - 플레이어 이름 유효성 검사와 포지션/티어 공백 검사를 통과한 경우 저장 버튼이 활성화된다. <br/> - 플레이어 이름이 중복된 경우, 중복되었다는 알림창을 띄우고 플레이어 이름 input에 포커스를 준다. <br/> - 저장 버튼을 클릭하면 넘어온 파라미터를 통해 홈화면으로 이동할지 매칭 화면으로 이동할지 결정한다. |

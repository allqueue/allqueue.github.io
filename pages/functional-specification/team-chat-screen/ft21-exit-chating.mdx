# 채팅 나가기 (FT21)

* 사용자를 체크한 뒤 채팅을 떠나는 것에 대해 사용자가 인지하기 힘들 수...
* 나가기 한 뒤 `팀원 평가` 하는 화면이 있다면 좋을 것 같다.

| ID | FT21 |
| --- | --- |

| 기능 이름 | 채팅 나가기 |
| 설명 | 현재 매칭된 채팅방을 나갈 수 있다. |
